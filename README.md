# 作業日報アプリ

放課後等デイサービスのパート従業員向けに、1日の作業を **15分単位・ワンタッチ** で記録できるWebアプリです。スマートフォン（iPhone）のブラウザで動作し、インストールやログインは不要です。

## 利用者の使い方

1. 配布されたURL（このリポジトリのGitHub PagesのURL）をSafariで開く
1. 共有ボタン →「ホーム画面に追加」でアイコンを置く
1. 以後はアイコンをタップして、作業項目を選び時間帯をタップするだけ

主な機能：

- 作業項目をタップ →「いまの時間に記録する」で現在の15分枠に記録
- タイムラインをタップして、まとめ入力・修正・取り消し
- 日報タブで合計時間を自動集計し、提出用テキストをコピー
- 設定タブで作業項目の追加・削除・並べ替え・名前変更、表示時間帯の変更

## データの保存について

- 記録は各利用者の端末内（ブラウザのローカルストレージ）にのみ保存されます
- このリポジトリやサーバーに記録データが送信されることはありません
- Safariの「履歴とWebサイトデータを消去」を行うと記録も消えるためご注意ください

## ファイル構成

|ファイル                  |役割                                    |
|----------------------|--------------------------------------|
|`index.html`          |アプリ本体（HTML/CSS/JavaScriptすべてを含む単一ファイル）|
|`apple-touch-icon.png`|ホーム画面に追加したときのアイコン（180×180）            |
|`icon-512.png`        |高解像度アイコン（予備）                          |

## 更新方法（管理者向け）

新しい `index.html` をこのリポジトリに上書きアップロード（Add file → Upload files）するだけで、URLはそのまま全利用者に反映されます。利用者側の作業は不要です。反映まで数分かかることがあります。
